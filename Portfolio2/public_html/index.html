HTML

HTML stands for Hyper Text Markup Language. The HTML language is made up of elements, which are then enclosed within tags. Tags are enclosed within angle brackets. Web browsers are able to read html files and compose them into visible web pages. Html was created by Tim Berners-Lee.

CSS

CSS, or Cascading Style Sheet, is a style sheet language that can be applied to most markup languages in order to make them look better. CSS was created by Opera Software's Haakon Wium Lie.

Bootstrap

Bootstrap is a free collection of tools for creating websites web applications. It was created by Mark Otto and Jacob Thornton. You can use HTML and CSS templates to create professional looking websites.

Javascript

Javascript is a computer programming language that allows user interaction with browsers. It is commonly used in video games as well as desktop and mobile applications. It was originally designed by Brendan Eich.

MelonJS

MelonJS is a free, open-source game engine that allows developers to more easily create video and computer games.
