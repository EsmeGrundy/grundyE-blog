Technology

HTML

HTML stands for Hyper Text Markup Language. The HTML language is made up of elements, which are then enclosed within tags. Tags are enclosed within angle brackets. Web browsers are able to read html files and compose them into visible web pages. Html was created by Tim Berners-Lee.

CSS

CSS, or Cascading Style Sheet, is a style sheet language that can be applied to most markup languages in order to make them look better. CSS was created by Opera Software's Haakon Wium Lie.
